SES3D Setup Assistant¶

This is a simple script that assists you in determining optimal SES3D settings. To use it just pass the number of elements and the size of the simulation domain.

$ python -m lasif.scripts.ses3d_setup_helper --help
usage: python -m lasif.scripts.ses3d_setup_helper [-h] dx dy dz nx ny nz

Script assisting in determining suitable SES3D settings.

positional arguments:
  dx          latitudinal extent in degrees
  dy          longitudinal extent in degrees
  dz          depth extent in km
  nx          elements in latitudinal direction
  ny          elements in longitudinal direction
  nz          elements in depth direction

optional arguments:
  -h, --help  show this help message and exit

Exemplary output is

$ python -m lasif.scripts.ses3d_setup_helper 30 47 1171 108 168 64
SES3D Setup Assistant

All calculations are done quick and dirty so take them with a grain of salt.

Extent in latitudinal (X) direction: 3339.6 km, 31 km/element
Extent in longitudinal (Y) direction: 5232.0 km, 31 km/element
Extent in depth (Z) direction: 1171.0 km , 18.3 km/element
